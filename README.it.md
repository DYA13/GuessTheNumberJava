# 🎯 Gioco Indovina il Numero

## 🚀 Come Giocare

1. Clona il progetto o scarica lo `.zip`
2. Assicurati di avere **Java (JDK)** installato (versione 8 o superiore)

### ▶️ Avvia il gioco

```bash
chmod +x run.sh     # solo la prima volta
./run.sh
```
